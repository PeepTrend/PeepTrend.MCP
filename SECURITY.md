# Security

Report security issues to support@peeptrend.com.

Please do not open public issues for secrets, bypasses, leaked keys, account takeover vectors, or other security-sensitive reports.

## API Keys

- Create keys only from your PeepTrend Profile.
- Store keys in your MCP client's environment configuration.
- Do not commit keys to git.
- Revoke keys immediately if they were shared or exposed.

PeepTrend stores API keys hashed and only shows a newly created key once.
